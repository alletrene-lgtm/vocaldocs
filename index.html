Comment utiliser ClipVoice

  1. 1 Sur une page web, sélectionnez un texte, appuyez longuement, puis choisissez Partager.
  2. 2 Dans la liste qui s'affiche, choisissez ClipVoice. Le texte arrive automatiquement.
  3. 3 ClipVoice lit le texte à voix haute. Appuyez sur Google Keep pour créer une note.
  4. 4 Alternative : copiez le texte, revenez ici, collez-le dans la zone ci-dessus, puis appuyez sur Utiliser ce texte.
