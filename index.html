📲 Installer sur Android (Chrome)

Cette application peut s'installer sur votre écran d'accueil, comme une vraie application.

  1. 1 Ouvrez cette page dans Chrome sur votre Android
  2. 2 Appuyez sur les 3 points ⋮ en haut à droite de Chrome
  3. 3 Cherchez « Ajouter à l'écran d'accueil » ou « Installer l'application »
  4. 4 Appuyez sur Installer ou Ajouter dans la boîte de dialogue
  5. 5 L'icône 🎧 apparaît sur votre écran d'accueil — l'appli fonctionne hors connexion
⚠️ Si vous ne voyez pas « Ajouter à l'écran d'accueil », vérifiez que vous utilisez bien Chrome (pas Samsung Internet, pas Firefox). L'option peut aussi s'appeler « Installer l'application » selon la version de Chrome.
📄→🎧

Utiliser avec Google Docs

Dans Google Docs → ⋮ → Partager et exporter → Envoyer une copie → Format texte, puis choisissez « Lecteur Vocal » dans la liste de partage Android (uniquement si l'app est installée).

Ou plus simplement : tout sélectionner → copier → coller dans l'onglet Texte.
